Diketahui dan ditanya:
1021+1022+1023+...+3018+3019+3020 = ?

Dijawab:
Langkah 1
Un = a + (n - 1)b
Un = 3020
3020 = 1021 + (n - 1)1
          = 1021 + (n - 1)
          = 1020 + n
       n = 3020 - 1020
          = 2000
Langkah 2
Sn = n/2 (a + Un)
S2000 = 2000/2(1021 + 3020)
            = 1000(4041)
            = 4041000 
            = 4041 x 10^6
